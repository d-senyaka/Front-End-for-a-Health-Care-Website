# Front-End for a Health Care Website

A clean, responsive **health care website front-end** built with **HTML, CSS, and JavaScript**.  
This project focuses on a modern UI/UX for a medical/health platform, including multiple pages such as a landing page, gallery, login/sign-up, reviews, and more.

## Getting Started (Run Locally)

### Option A — Open directly
1. Download / clone the repository
2. Open `index.html` in your browser

### Option B — Use a local server (recommended)
Using VS Code:
1. Install the **Live Server** extension
2. Right-click `index.html` → **Open with Live Server**

Or using Python:
```bash
# Python 3
python -m http.server 5500
```
Then visit:
- http://localhost:5500
